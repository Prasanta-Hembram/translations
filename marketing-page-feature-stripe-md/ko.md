앱 내에서 카드로 즉시 결제하고 싶으시다면 Stripe를 이용하세요. 다음 항목에 '카드' 결제 수단을 설정할 수 있습니다 :

- 회의실 예약 페이지.
- 인보이스 결제 페이지.

Stripe 결제는 다른 앱을 설치하거나 등록하지 않아도 앱 내부에서 작동합니다. Stripe를 연동하시려면 API 키를 준비하시면 됩니다 : https://dashboard.stripe.com/apikeys
